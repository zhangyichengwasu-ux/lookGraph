## 1. 账户管理

**Controller**: `EnterpriseAccountController`  
**路径前缀**: `/enterprise/account`  
**业务含义**: 管理企业账户的详情查询、交易明细、账户激活和挂帐状态查询。

| 方法 | 路径 | 业务含义 |
|------|------|----------|
| GET | `/enterprise/account/detail` | 查询当前企业账户的详情，包括余额、账户状态等核心信息 |
| GET | `/enterprise/account/scene/detail` | 查询场景子户账户详情，用于分场景查看账户资金情况 |
| GET | `/enterprise/account/record` | 分页查询企业账户的交易明细记录（充值、消费、退款等流水） |
| GET | `/enterprise/account/export` | 导出企业账户交易明细为 Excel 文件 |
| GET | `/enterprise/account/activat/link` | 获取激活企业账户的链接，用于新账户激活流程 |
| GET | `/enterprise/account/credit/status` | 查询挂帐账户的当前状态（是否开通、额度等） |

| GET | `/enterprise/credit/transaction/download/excel` | 导出额度明细为 Excel 文件 |
| GET | `/enterprise/credit/order/pending/page` | 待还款订单分页列表，展示需要还款的订单 |

## 4. AI 多轮会话

**Controller**: `ChatController`  
**路径前缀**: `/chat`  
**业务含义**: 核心 AI 对话功能，支持流式对话（SSE）、历史会话管理、商品搜索、表格编辑、文件上传等。用户通过与 AI 多轮会话完成商品检索和比价。

| 方法 | 路径 | 业务含义 |
|------|------|----------|
| POST | `/chat/stream` | **流式对话（SSE）**：发送对话消息，通过 SSE 实时接收 AI 回复（支持文本、思考过程、商品推荐三种消息类型） |
| GET | `/chat/resume` | 恢复未结束的会话内容（SSE），从 Redis 缓存读取未推送完的数据继续推送 |
| POST | `/chat/edit_stream` | 编辑重新生成（SSE），取消当前活跃任务，覆盖原提问内容重新生成回答 |
| POST | `/chat/history` | 分页查询历史会话列表，按创建时间倒序排列 |
| POST | `/chat/rename` | 重命名历史会话标题 |
| POST | `/chat/delete` | 删除指定历史会话 |
| POST | `/chat/top` | 置顶或取消置顶会话 |
| POST | `/chat/create` | 创建新的 AI 对话会话 |
| GET | `/chat/messages/byRound` | 查询会话消息列表（按轮次分组），返回二维列表展示问答轮次 |
| POST | `/chat/messages/byRound/page` | 分页查询会话消息（按轮次分组），支持大量消息的翻页加载 |
| POST | `/chat/stage/group/skus` | 根据阶段ID和分组名称分页查询商品列表，用于 AI 分组推荐的商品详情查看 |
| POST | `/chat/upload/file` | 上传对话文件（图片或文件），用于多模态对话输入 |
| GET | `/chat/file/previewUrl` | 获取文件云存储地址的预签名预览 URL |
| POST | `/chat/stage/group/filter` | 获取 AI 对话阶段商品分组的扩展属性和品牌筛选项 |
| POST | `/chat/stage/table/filter` | 获取表格搜索商品的扩展属性和品牌筛选项 |
| POST | `/chat/stage/group/skuFilterPage` | 根据筛选条件（城市、价格区间、品牌、规格等）对分组内商品进行筛选和排序分页 |
| POST | `/chat/stage/table/skuFilterPage` | 对表格阶段的全部商品进行筛选排序分页查询 |
| POST | `/chat/stage/id` | 根据 chatId、answerId、groupName 获取阶段ID |
| POST | `/chat/table/cell/edit` | 表格单元格重新编辑搜索，修改表格中某个单元格的商品搜索条件重新搜索 |
| POST | `/chat/stage/table` | 根据 stageId 分页获取表格阶段中所有单元格的内容 |
| GET | `/chat/stage/group` | 根据阶段ID和分组名获取分组阶段的媒体内容（仅包含指定分组） |

| POST | `/order/cancel` | 取消指定订单，释放锁定的库存和额度 |

| GET | `/order/detail/{orderId}` | 查询订单详情，包括商品明细、金额、配送信息、支付状态等 |
| GET | `/order/page` | 个人中心-我的订单分页列表，支持按状态筛选 |
| GET | `/order/down` | 下载订单列表数据，提交异步任务到下载中心 |
| GET | `/order/count` | 统计指定城市下不同状态的订单数量（待付款、待发货等） |
| POST | `/order/pay/balance` | **余额支付**：使用企业账户余额支付订单（幂等接口） |
| POST | `/order/pay/credit` | **挂帐支付**：使用企业挂帐额度支付订单（幂等接口） |
| POST | `/order/pay/sms-code/{orderId}` | 支付前获取短信验证码，用于支付安全验证 |
| GET | `/order/pay/result/{orderId}` | 轮询查询支付结果，用于异步支付流程中的状态确认 |

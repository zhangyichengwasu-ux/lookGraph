## 1. 账户管理

### 1.1 企业账户（EnterpriseAccountController）

**路径前缀**：`/enterprise/account`  
**业务含义**：管理企业用户在商城端的资金账户，包括账户详情查看、交易明细查询、明细导出、账户激活、挂帐状态查询等。

| 方法 | 路径 | 接口说明 | 业务含义 |
|------|------|----------|----------|

## 2. AI 多轮会话

### 2.1 多轮会话（ChatController）

**路径前缀**：`/chat`  
**业务含义**：AI 智能采购助手的核心接口，支持通过多轮对话方式进行商品搜索、筛选和对比。用户可通过文字、图片、文件等方式描述采购需求，AI 理解意图后返回商品推荐。支持流式对话（SSE）、历史会话管理、商品分组展示、表格对比等功能。

| 方法 | 路径 | 接口说明 | 业务含义 |
|------|------|----------|----------|
| POST | `/stream` | 流式对话（SSE） | 发起与AI采购助手的流式对话，用户输入采购需求，AI实时返回商品推荐（支持文字、思考消息、商品查询消息三种SSE事件类型） |
| GET | `/resume` | 恢复未结束的会话（SSE） | 断线重连时从Redis缓存中恢复未完成推送的会话内容 |
| POST | `/edit_stream` | 编辑重新生成（SSE） | 修改已发送的提问内容，取消当前回答并重新生成 |
| POST | `/history` | 历史会话列表（分页） | 分页查询用户的所有历史对话记录，按创建时间倒序 |
| POST | `/rename` | 历史会话重命名 | 修改指定会话的标题名称 |
| POST | `/delete` | 删除历史会话 | 删除指定的历史会话及其所有消息 |
| POST | `/top` | 置顶/取消置顶会话 | 将指定会话置顶或取消置顶 |
| POST | `/create` | 创建新会话 | 创建一个新的空白AI对话会话 |
| GET | `/messages/byRound` | 按轮次查询会话消息 | 查询指定会话的所有消息，按对话轮次分组（一轮 = 用户提问 + AI回答） |
| POST | `/messages/byRound/page` | 按轮次分页查询会话消息 | 分页查询指定会话的消息，按轮次分组 |
| POST | `/stage/group/skus` | 根据阶段ID和分组名分页查询商品 | 查询AI回答中某个分组下的商品列表（如"推荐品牌A的商品"分组） |
| POST | `/upload/file` | 上传对话文件 | 上传图片或文件作为AI对话的输入素材 |
| GET | `/file/previewUrl` | 获取文件预览地址 | 根据云存储路径获取预签名预览URL |
| POST | `/stage/group/filter` | 获取商品的扩展属性和品牌 | 获取AI对话分组中商品的筛选条件（品牌、规格等） |
| POST | `/stage/table/filter` | 获取表格搜索筛选项 | 获取AI表格搜索结果中商品的筛选条件 |
| POST | `/stage/group/skuFilterPage` | 根据筛选条件分页查询分组商品 | 对AI推荐的某个分组商品按品牌、价格、规格等条件二次筛选 |
| POST | `/stage/table/skuFilterPage` | 表格商品查询全部 | 对AI表格搜索结果的商品进行筛选排序分页 |
| POST | `/stage/id` | 获取阶段ID | 根据会话ID、回答ID、分组名获取对应的阶段ID |
| POST | `/table/cell/edit` | 表格单元格重新编辑 | 修改AI搜索表格中某个单元格的搜索条件并重新查询 |
| POST | `/stage/table` | 分页获取表格阶段内容 | 分页查询AI搜索表格阶段的单元格数据 |
| GET | `/stage/group` | 获取分组阶段媒体内容 | 根据阶段ID和分组名获取该分组的完整推荐结果 |

### 关键技术特征

- **认证方式**：通过 `BaseController` 和 `UserContext` 获取当前登录用户信息
- **幂等控制**：支付、创建还款等关键操作使用 `@Idempotent` 注解防重复提交
- **流式响应**：AI 对话接口使用 SSE（Server-Sent Events）实现流式响应
- **异步任务**：下载、导出等耗时操作通过异步任务处理，结果在下载中心获取
- **文件上传**：支持 MultipartFile 上传图片、Excel 等文件
- **分页查询**：统一使用 `PageVO<T>` 或 `PageResponse<T>` 封装分页结果
- **城市隔离**：大部分业务接口按城市（cityId）进行数据隔离
